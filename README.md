### 🧠 About me

I build AI systems that solve real problems — computer vision pipelines, voice-driven assistants, and cloud-deployed ML apps that actually work in the wild.

My projects start with a meaningful problem (drowsy drivers, untracked attendance, late-night debugging sessions) and end with a production-ready system where ML handles what rule-based logic never could.
