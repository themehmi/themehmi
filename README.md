I'm a developer obsessed with the intersection of **computer vision**, **voice AI**, and **intelligent systems** that actually do something useful in the real world.

My projects tend to start with a problem worth solving — drowsy drivers, untracked attendance, late-night debugging sessions — and end with a deployed, production-ready system that uses ML to handle what rule-based logic never could.

### 🧠 What I build

- **AI-powered assistants** — voice interfaces backed by LLMs (Llama 3.1 via NVIDIA NIM), real-time alert systems, and multimodal pipelines that combine sight and sound
- **Computer vision systems** — face recognition, EAR-based drowsiness detection, DeepFace-powered attendance tracking
- **Full-stack ML apps** — Flask backends, MongoDB Atlas, Hugging Face Spaces deployments, cloud-native from day one
